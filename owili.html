Owili

Mobile App Wireframes

Owili is an app for a theoretical self-driving wheelchair. This was a class project where we conducted user research, created personas, identified user goals, used competitive analysis, referenced visual languages of similar products, recorded user tests, and designed high-static mockups.

Our visual language references included Google Maps, Uber, Lyft, Volvo, and Tile. We pulled the successful parts of each of these applications to integrate into our own application. It targets anyone with a disability that requires a wheelchair and is often moving back and forth from their chair to a separate activity. Essentially the wheelchair would locate the user through GPS and drive to them when they need it.
