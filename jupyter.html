Website Navigation Bar

One of our first projects was to redesign the navigation on the main website. The problem was the original navigation bar had 8 menu items already filling up most of the horizontal space and the team wanted to add more menu items. We looked at google analytics to analyze which pages were receiving the most traffic and we discussed with our boss which menu items we wanted to increase traffic to. Using this information, we ordered the menu items based upon importance. Some of the original menu item names were misleading as well and we renamed them to be more intuitive.

As a team we brainstormed solutions and decided on nested menus. We mocked-up some different versions of the nested menus by grouping different menu items together and user tested them to see which ones were most intuitive. Once we finalized our results, we created a final mockup to be implemented.


JupyterLab

Most of our projects were focused on improving JupyterLab. We were given various widgets to work on and improve throughout the summer. Some of them included an Image Widget, a FAQ Page, and JupyterLab’s launcher menu. We executed a design driven development process as a team. Each of these widgets required creating user profiles and studying which key features to include in the widget that would accommodate the user best. We conducted user tests to refine our results and finalize the final design.
